Main content

  left:75%	
}

#container-outer {
  float:left;
  background: #eee;
}

The key to this layout is that for each column in the layout there's a corresponding container div. When both are floated the container and column will have the same height.

As with the other equal height column methods We set background colors on the containers. Because all the container divs will have the same height it will appear as though the columns are of equal height.

We use relative positioning to shift both container divs and column divs into place.
